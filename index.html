ToDo Task list

This project involves creating a simple to-do task list application that allows users to input tasks and store them using local storage. The application has a user-friendly interface with easy-to-use functions.
  1. Add tasks: Users can input tasks into the application using a text input field and a "Add Task" button
  2. Edit tasks: Users can edit a task and save the changes to both the innerHTML and local storage.
  3. Remove tasks: Users can delete tasks by clicking on the "Delete" button associated with each task.
  4. Reset all tasks: Users can clear all the listed tasks by clicking the "Reset All" button.
  1. The application is built using HTML, CSS, and JavaScript.
  2. The tasks entered by users will be stored using local storage.
  3. The application uses event listeners to capture user input and perform the necessary actions (add, delete, edit, reset.).
  4. The application uses a responsive design to ensure that it works well on different screen sizes.
  1. User-friendly interface: The application has a clean and simple interface that is easy to use.
  2. Local storage: The application uses local storage to store the tasks entered by users, ensuring that the tasks are saved even if the user closes or refreshes the application.
  3. Task manipulation: The application allows users to easily add, delete, edit and clear tasks.
  4. Reset and clear tasks: The application allows users to clear all tasks at once with a button click which removes from local storage also.
